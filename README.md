# Проектирование Arduino Uno в Altium Designer

Репозиторий содержит материалы учебного цикла из 8 лабораторных работ по курсу **«Конструирование приборов»**. В рамках проекта осуществляется воссоздание платы **Arduino Uno** с адаптацией под стандарты **ГОСТ/ЕСКД**.

## 🎯 Цель проекта
Пройти полный цикл проектирования электронного узла в САПР Altium Designer: от создания библиотек компонентов до генерации производственных файлов.

## 🛠 Технологический стек
* **САПР:** Altium Designer
* **Стандарты:** ГОСТ 2.701-2008, ГОСТ 2.106-1996
* **Компонентная база:** ATMEGA328P-PU и сопутствующая обвязка
